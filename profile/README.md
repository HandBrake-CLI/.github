## Fixing Common Command Issues

Command not found: confirm the HandBrake CLI install location, update the system path, and reopen the terminal before running HandBrake CLI commands.  
Preset missing: list available HandBrake CLI presets and compare names exactly before reusing older scripts.  
Subtitle mismatch: test HandBrake CLI subtitles on a short clip, then adjust track selection, burn-in behavior, or language options.  
Batch output errors: check folder permissions, free disk space, and filename patterns before repeating a HandBrake CLI batch convert job.
